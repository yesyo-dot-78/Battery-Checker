# Battery Checker
***THIS ONLY WORKS FOR WINDOWS MACHINES***

A straight beep sound *(eeeee sound)* means that battery is higher than ***80%*** and charger is on

An ICONEXCLAMATION sound *(ding sound)* means that the battery is lower than ***30%*** and charger is off
